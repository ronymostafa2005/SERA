# منصة الإسلام الشاملة

مشروع Next.js لتطبيق إسلامي شامل يحتوي على:
- القرآن الكريم مع تلاوات صوتية
- قصص الأنبياء
- محاضرات إسلامية
- أدعية وأذكار
- ملف شخصي للمستخدم

## التقنيات المستخدمة

- **Next.js 15** - إطار عمل React
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS** - للتصميم
- **React Context API** - لإدارة الحالة
- **Heroicons** - للأيقونات

## البدء

### تثبيت التبعيات

```bash
npm install
```

### تشغيل المشروع في وضع التطوير

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

### بناء المشروع للإنتاج

```bash
npm run build
```

### تشغيل المشروع بعد البناء

```bash
npm start
```

## البنية

```
src/
├── app/              # صفحات Next.js (App Router)
│   ├── page.tsx      # الصفحة الرئيسية
│   ├── login/        # صفحة تسجيل الدخول
│   ├── signup/       # صفحة التسجيل
│   ├── dashboard/    # لوحة التحكم
│   ├── quran/        # القرآن الكريم
│   ├── stories/      # قصص الأنبياء
│   ├── lectures/     # محاضرات إسلامية
│   ├── duas/         # أدعية وأذكار
│   └── profile/       # الملف الشخصي
├── Components/       # المكونات المشتركة
├── Context/          # Context API
├── Pages/            # صفحات React
├── assets/           # الصور والملفات الثابتة
└── utils/            # وظائف مساعدة
```
